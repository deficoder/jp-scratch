### 日语学习规划: 从零基础到N2水平

#### 第一阶段 绝对入门(0-3个月)

**核心目标** 掌握五十音, 学会最基本的问候与句型

**关键资源** 井上一宏老师的[从零开始学日文系列](https://www.youtube.com/playlist?list=PLzvvBBq1AJaFR13DEbJpjnmMgybkmLMcH)

**时间预估** 每天1-1.5小时, 持续约3个月

#### 第二阶段 基础巩固(第4-9个月)

**核心目标** 系统学习初级语法与词汇, 目标达到N4-N3水平

**关键资源**

- 系统课程 继续跟随井上一宏的[进阶内容](https://www.youtube.com/playlist?list=PLzvvBBq1AJaEedLMiQHkNM4YBz_-RcnxR)或《大家的日语》

- 辅助频道 [Japanese Ammo with Misa](https://www.youtube.com/@JapaneseAmmowithMisa)(讲解细致)、[NIHONGONOMORI](https://www.youtube.com/@nihongonomori2013)(系统性强, 适合备考)

**时间预估** 每天1.5-2小时, 持续约6个月

#### 备考冲刺(第10-18个月)

**核心目标** 掌握N2级别语法与词汇, 进行大量真题训练

**关键资源**

- 备考专项 NIHONGONOMORI有针对JLPT各等级的系统备考视频

- 听力强化 [Easy Languages](https://www.youtube.com/@easylanguages)的街头采访和[Onomappu](https://www.youtube.com/@Onomappu)的生活化视频, 能帮你适应真实语速

**时间预估** 每天2-3小时, 持续约8-9个月
